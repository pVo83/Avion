# Avion

Avion — интернет-магазин мебели и товаров для дома, разработанный на Vue 3. Проект включает главную страницу, страницу о нас, каталог товаров, страницу продукта, корзину и адаптивные layout-компоненты.

## Стек

- Vue 3
- Vite
- Vue Router
- Pinia
- SCSS
- ESLint / Stylelint

## Особенности

- Адаптивная верстка под desktop, tablet и mobile.
- Переиспользуемые UI-компоненты: кнопки, логотип, навигация, бургер, мобильное меню, карточки товаров.
- Мобильное меню с overlay, анимацией через `<Transition>` и управлением состояния через Pinia store.
- Корзина товаров с управлением количества и итоговой суммы.

## Команды

```bash
npm install
npm run dev
npm run build
```

Для проверки и автоисправления кода:

```bash
npm run lint
npm run stylelint:fix
```
